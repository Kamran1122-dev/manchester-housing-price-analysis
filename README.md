# Manchester Housing Price Analysis

This project analyzes housing prices in Manchester using Python.
It focuses on understanding how property features such as bedrooms, area size, and location (postcode) impact price variation.

### Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

### Key Insights
- Property prices differ significantly by postcode.
- Larger houses and properties with more bedrooms tend to have higher prices.
- Detached properties show higher price ranges compared to flats and terraced houses.
